# AI 咖啡店新品宣傳 Skill

## 專案介紹
這是一個利用 AI 協助咖啡店製作新品宣傳內容的工作流程。

使用者只需要輸入新品名稱、特色、價格與活動時間，
AI 就可以協助產生適合社群平台使用的宣傳文案。

## 工作流程

1. 輸入新品名稱
2. 輸入商品特色
3. 輸入價格與活動時間
4. AI 產生宣傳文案
5. 將文案用於 IG、Facebook 或店內公告

## 使用工具

- ChatGPT
- GitHub
- GitHub Pages

## 適用情境

- 咖啡店新品上市
- 飲料店季節限定商品
- 校園市集攤位宣傳
- 小型品牌社群經營

咖啡店推出期間限定飲品，結合蜜桃果香、氣泡水與濃縮咖啡，適合夏天飲用。
